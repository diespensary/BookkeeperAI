# Название

MVP "Телеграмм-бот для учета расходов с голосовым вводом и ИИ"

# Описание

- Можно вводить траты текстом или голосовым сообщением
- Главная фича: не нужно придерживаться определенного шаблона при вводе траты (ИИ сам все распарсит)

# Запуск

1. `git clone`
2. `export TELEGRAM_BOT_TOKEN=`
3. `export HUGGINGFACE_TOKEN=`
4. `./gradlew clean bootRun`
